About the Project

This project demonstrates how to automatically append professional email signatures to outgoing emails using Amazon WorkMail and AWS Lambda. The solution dynamically selects and injects signatures based on contextual rules such as sender address or department, ensuring consistent branding, compliance messaging, and professional communication across the organization.
