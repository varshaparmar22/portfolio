About the Project

Designed and implemented a fully automated static website deployment pipeline using AWS native services integrated with GitHub. This project demonstrates end-to-end DevOps automation, serverless architecture, and secure global delivery ‚Äî all built within the AWS Free Tier.

Features

  • Continuous Deployment: Automatic deployment from GitHub ‚Üí AWS S3 on every push to the main branch via AWS CodePipeline.
  • Zero Downtime Delivery: Uses Amazon S3 static hosting with version-safe rollbacks.
  • Global Performance: Integrated Amazon CloudFront CDN to cache assets globally and serve the site over HTTPS with minimal latency.
  • Smart Cache Invalidation: An AWS Lambda function automatically invalidates CloudFront cache whenever new content is deployed.
  • Event-Driven Automation: Amazon EventBridge triggers the Lambda upon CodePipeline SUCCEEDED events ‚Äî a 100% serverless CI/CD flow.
  • Secure Integration: GitHub connected using AWS CodeStar Connections, ensuring secret-free deployments without exposing credentials.
  • Cost-Effective: Utilizes AWS Free Tier services, with no additional costs for static hosting or CDN.

  • 1. Developer commits changes to main in GitHub.
  • 2. AWS CodePipeline automatically builds and deploys updated files to S3.
  • 3. EventBridge detects pipeline success and invokes a Lambda function.
  • 4. Lambda invalidates CloudFront cache to ensure users instantly get fresh content.
  • 5. CloudFront globally serves the latest version through its CDN edge locations.
  • 6. API Gateway receives the request and forwards it to the Lambda function.
  • 7. Lambda function processes the request and sends an email using AWS SES.
  • 8. SES sends the email to the specified recipient.
